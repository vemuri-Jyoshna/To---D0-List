To-Do List 📝

A simple and interactive To-Do List Application built using HTML, CSS, and JavaScript. This project helps users organize daily tasks by adding, completing, and deleting tasks easily.


🚀 Features

- Add new tasks

- Mark tasks as completed

- Delete tasks

- Responsive and clean UI

- Real-time task management

- Easy to use



🛠️ Technologies Used

- HTML5

- CSS3

- JavaScript


🎯 How It Works


1. Enter a task in the input field.


2. Click the Add button.


3. View tasks in the task list.


4. Mark tasks as completed or delete them when finished
